# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and inclusive environment for everyone participating in this project.

Participants are expected to interact professionally and respectfully in all project spaces, including:

- Issues
- Pull Requests
- Discussions
- Code reviews
- Community conversations
- Events or meetings related to the project

## Expected Behavior

Examples of behavior that contribute to a positive environment include:

- Being respectful and considerate
- Providing constructive feedback
- Collaborating openly and professionally
- Respecting differing viewpoints and experiences
- Showing empathy toward other community members

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment or discriminatory language
- Personal attacks or insults
- Trolling or inflammatory comments
- Publishing private information without permission
- Any conduct that could reasonably be considered inappropriate in a professional setting

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may remove, edit, or reject comments, commits, issues, pull requests, or other contributions that violate these guidelines.

## Scope

This Code of Conduct applies to all interactions within the project community and in public spaces when an individual is representing the project.

## Reporting

If you experience or witness unacceptable behavior, please report it to the project maintainers.

## Enforcement

Project maintainers may take appropriate action in response to violations of this Code of Conduct, including warnings, temporary bans, or permanent removal from the community.

## Acknowledgment

This Code of Conduct is inspired by the Contributor Covenant Code of Conduct.
